# Сервер обновления контента

Сервер отдаёт JSON с актуальными заданиями для мобильного приложения.
Он использует только стандартную библиотеку Python.

## Запуск

```powershell
python server.py --host 0.0.0.0 --port 8000
```

В Android-эмуляторе адрес сервера указывается как:

```text
http://10.0.2.2:8000
```

Основные эндпоинты:

- `GET /api/health` — проверка доступности сервера.
- `GET /api/content` — полный набор данных для обновления SQLite.
- `GET /api/updates` — алиас для получения обновлений.
